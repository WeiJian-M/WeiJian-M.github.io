Latex02: 中文的处理方法

% 导言区
\documentclass{article} %article, book, report, letter(letter 不使用 \maketitle)

\usepackage{ctex} % 在cmd命令行窗口输入 `texdoc ctex` 可打开ctex文档

\newcommand\degree{^\circ} % 角度符号

\title{\heiti 我的第一个文档}
\author{\kaishu 小猪佩奇}
\date{\today}

% 正文区（文稿区）
\begin{document} % 一个latex文件（正文区）有且仅有一个document文件
    \maketitle % 输出标题
    勾股定理可以用现代语言表述如下:

    直角三角形斜边的平方等于西腰的平方和。

    可以用符号语言表述为:设直角三角形$ABC$， 其中$\angle C=90\degree$，则有:
    \begin{equation} % 用于生成带编号的公式
        AB^2 = BC^2 + AC^2.
    \end{equation}

\end{document}

  • 为了显示中文，我们需要在导言区导入ctex包：\usepackage{ctex}；

  • 在文字前面可以指定字体类型，比如使用\title{\heiti 我的第一个文档}使标题显示为黑体，使用\author{\kaishu 小猪佩奇}使作者名显示为楷书

  • 此外，如果想要在文中显示带编号的公式，需要使用如下格式：

\begin{equation} % 用于生成带编号的公式
        你的公式
\end{equation}
